Lift 3 Demo

Welcome to the Lift 3 Demo at

This fully functional example application demonstrates some of the new and exciting features of Lift 3, such as:

  • Markdown support
  • Futures support in REST and Snippets
  • Roundtrip Promises
  • Actors across address space
  • Custom Data Attribute support

More Info

You can clone this project from https://github.com/tuhlmann/lift-3-demo

It uses the lift-bootstrap.g8 example template and the MapperAuth liftmodule for authentication and role/permission management against SQL databases.
